Plingnote Release Notes

Release 2

In this version you can:

  1. Add/edit notes
  2. View notes as a list
  3. View notes in a grid
  4. View notes as markers on the map
  5. Delete notes
  6. Search for notes
  7. Clear search history trough settings
  8. Send feedback to us developers
  9. View application information such as version, licenses and terms of use.

Changelog

  1. The notes can now be viewed as markers on a map.
  2. Selected notes in the list are now highlighted.
  3. User can now see the date of when the note was created in the list view.
  4. A third tab is now available, where you can see notes in a grid

Known bugs and limitations

  1. When rotating the screen while items are selected in the list view, the contextual action menu disappears and the items are unselected.
  2. Search bar is closed when rotating the device.
  3. The database does not back up data when upgraded.
  4. Notes with specified images will not be shown in the gridtab, only standard images are shown.
