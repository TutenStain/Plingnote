Plingnote Release Notes

Release 3

This document provides version-specific information about the Plingnote application.

In this version you can:

  1. Add/edit notes
  2. View notes as a list
  3. View notes in a grid
  4. View notes as markers on the map
  5. Delete notes
  6. Search for notes
  7. Clear search history trough settings
  8. Send feedback to us developers
  9. View application information such as version, licenses and terms of use.
  10. Add an image to the note.
  11. Fit all added notes so that they are all visible at the same time.
  12. Easily see if the notes has an image or alarm attached in the list view.

Changelog

  1. Users can make all added notes visible on the map, without the need to move around.
  2. If an alarm or image is attached to a note, a small icon will appear on the specific item in the list view.

Known bugs and limitations

  1. When rotating the screen while items are selected in the list view, the contextual action menu disappears and the items are unselected.
  2. Text in search bar is not saved when rotating the screen.
  3. The database does not back up data when upgraded.

Getting started

When you first start the application there will be (from top to bottom) an options menu, tabs and a map where you can view added or notes or add new ones.

Adding a note.

To add a note, press the “+” button in the options menu and you will be directed to the editing mode. Apart from adding title and text, you might’ve noticed that you can add more things to the note at the bottom.

Location

Clicking the location alternative will directed you to the mapview where you will be able to set a location for the note.

Reminder

Add a reminder to your note.

Category

You can choose between 7 different categories to add to your note.

Viewing your note

When you are done editing you note, just press back or press the icon in the upper left corner and you will be directed back to the map. If there’s a location added to the note it will be visible on the map. If you didn’t add a location to your note, don’t worry! The note still will be viewable in the list and grid view.

Search through your notes

To search through added notes, simply press the magnifier at the top of the screen. Be careful though - if you search on something usual you will probably multiple results.

Settings

When you have a large search history or just want to send feedback to us about your experience with the application - hit the options button on your device. There you’ll be able to, for example, clear the search history and send feedback to us!
