Plingnote

User Manual

Plingnote is an Android application in which you can write notes and, if desired, assign them to a specific location on a map. Apart from adding title and content, it’s also possible to set an alarm to be triggered when the user is approaching the location assigned to a note.

Views

Plingnote offers three different ways of view saved notes - in a list, in a grid or on a map - where each view has its own benefits. In all views, new notes can be added by pressing the “+”-button in the upper right corner. To navigate between the views, you can press the tabs at the top of the screen. Another way of navigating between them is to swipe from the sides.

TRE STYCKEN BILDER PÅ SAMTLIGA VYER

Map

In the map view, it’s possible to view saved notes on a map. To zoom, simply move your thumb and finger in a pinch gesture. Use your fingers to drag the map to the desired position. Tap anywhere on the map to add a new note.

List

The list view offers an ordinary list of notes, ordered by creation date. To edit or read a saved note, click in the desired column. A great feature of the list view is the ability to delete multiple items. To do so, hold your finger on any note until a menu is shown at the top of the screen. You delete the selected items by clicking the trash bin icon to the right. There are several ways of dismissing the menu; deselect all the notes, change view or click the done button to close it manually. You can also search for a specific note by clicking the magnifier.

Grid

In the grid view, the notes are shown as thumbnails. Depending on what category the note belongs to, an image will tell what category it belongs to. Notes can be deleted in the same way as in the list view.

Edit note

In this view, you can read or edit your note. Simply tap on field you want to change. The title of the note is entered in the top field, while the actual content of the note is entered below.

Additional note content

The bottom of the screen consists of a bar where you can add additional content to the view, such as picture, category, alarm and location. You can swipe to switch between them.

BILDER PÅ SAMTLIGA SNOTEBARVYER

Picture

Let’s you pick a picture from your device’s gallery and assign it to the note.

Category

Add a category to your note to make it easier to find it.

Alarm

Let’s you set an alarm to the note. The alarm is triggered when you’re approaching the location of the note.

Location

Add a location to the note to make it visible on the map.

Settings

To launch the settings view, click the icon with three dots in the upper right corner. It consists of two sections, Search and About.

Clear Search History

Let’s you clear the search history so that no recent entries are displayed when typing into the search field.
