Plingnote Developer’s manual

  1. Fetch the git repository from https://github.com/TutenStain/Plingnote
  2. Make sure you have the Android SDK, an Android (Virtual) Device and Java SE development environment.
  3. To build a runnable APK either use ANT with the command “ant release” or build it via eclipse.

The Package structure is a simple all-in one structure and will not be changed until the next release.

Two Layers

The app

The application layer adopts the standard design pattern for Android. Activities/fragments that take care of input/output logic and transitions between activities/fragments. Data that is saved is fetched from the DataProvider.

DataProvider

This layer provides content to the activities. It contains an SQLite database for local data. The database can be listened to and will notify its observers what actually happened by sending an enum type.
